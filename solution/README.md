Instructions
==========

## First thing:

### Clone the repository

```
  $ git clone https://github.com/Vitormdias/desafioTegra.git
  $ cd desafioTegra
```

## How to Use:

Run the Main program:

```
  $ python3 pair_programming_draft.py
```

## How to Test

There are also some small automatized tests

Just run:

```
  $ python3 run_tests.py
```
